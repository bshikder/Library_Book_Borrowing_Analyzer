# 📚 Library Book Borrowing Analyzer

## 🔍 Project Overview

The **Library Book Borrowing Analyzer** is a data analysis project developed using Python and Jupyter Notebook. This project explores borrowing patterns within a library system to uncover trends, user behavior, and popular book categories.

The project demonstrates practical skills in data analysis, visualization, and the use of AI-assisted coding tools such as Codex.

---

## 🎯 Objectives

* Analyze book borrowing patterns
* Identify popular books and categories
* Understand user borrowing behavior
* Visualize insights using charts and graphs

---

## 🛠️ Technologies Used

* **Python**
* **Jupyter Notebook**
* **Pandas** (Data manipulation)
* **Matplotlib** (Data visualization)
* **Codex (AI Assistance)**

---

## 📊 Key Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Bar chart visualization for category analysis
* Histogram visualization for borrowing distribution
* Insight generation from dataset

## 🚀 How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/bshikder/Library-Book-Borrowing-Analyzer.git
   ```

2. Navigate to the project folder:

   ```bash
   cd Library-Book-Borrowing-Analyzer
   ```

3. Open the notebook:

   ```bash
   jupyter notebook
   ```

4. Run all cells to see analysis and visualizations.
